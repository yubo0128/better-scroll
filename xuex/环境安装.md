跟新

```
sudo apt-get update
```

安装yum

```
apt install yum
```

下载nginx包

```
wget http://nginx.org/download/nginx-1.14.2.tar.gz
```

解压包

```
tar -xzf 包名称
```

进入文件

```
cd  nginx-1.14.2
```

执行配置文件

```
./configure
```

默认会安装到 /usr/local/ 目录里

编译安装命令

```
make && make install
```

Ubuntu

```
apt-get -y install gcc automake autoconf libtool make

```
